# Raycasting

This game simulwated 3d graphics, using raycasting algorithm.

![Gameplay, in dev]()

## Quitck start
```bash
# clone repo
git clone https://github.com/mental0-main/raycasting.git

# move to dir
cd raycasting

# build
mkdir build && cd build
cmake .. && make -j4

# run
./raycasting
```

## Controls
| Input | Acation |
-------------------
| WASD | Movement |
| Mouse motion | Look around |
| Escape | Quit |
